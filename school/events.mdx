---
title: "Creating and Managing School Events and Sending Notices"
description: "Create and manage school events in Elimu Bora — from term openings and holidays to weekly assemblies — and automatically notify the right people."
---

Events in Elimu Bora give you a central place to plan and communicate what's happening at your school. Whether you're announcing a sports day to the whole school or sending an exam-week reminder to a specific class, the events module handles the scheduling, targeting, and notifications for you.

## Event types

Every event you create belongs to one of five categories:

| Type | When to use it |
|---|---|
| **General** | Assemblies, sports days, parent meetings, and any other school activities |
| **Term Opening** | The first day of each term |
| **Term Closing** | The last day of each term |
| **Mid-Term Break** | Mid-term breaks between sessions |
| **Holiday** | National holidays and school closures |

## Create an event

<Steps>
  <Step title="Open the Events section">
    From the main navigation, go to **Events**, then click **New Event** in the top right.
  </Step>
  <Step title="Fill in the event details">
    Enter the event **title**, select a **date and time**, choose an **event type**, and add a **description** to give attendees context.
  </Step>
  <Step title="Set the audience">
    Choose who should see this event:

    - **School-wide** — tick the "School-Wide Event" toggle to notify everyone at the school.
    - **Targeted** — leave the toggle off and manually select attendees. You can add individual students, teachers, guardians, entire grade levels, or specific grade streams.
  </Step>
  <Step title="Set up recurrence (optional)">
    If the event repeats, enable the recurrence option and choose **weekly** or **monthly**. For example, set a weekly assembly to repeat every Monday so you only have to create it once.
  </Step>
  <Step title="Add attachments (optional)">
    Link related resources to the event:

    - **Assessments** — attach an exam to an exam-week event so attendees can see what's coming up.
    - **Invoices** — attach a fee invoice to a payment day event.
    - **Documents and circulars** — upload PDFs, letters, or programmes directly to the event.
  </Step>
  <Step title="Save the event">
    Click **Save**. Elimu Bora will immediately send an email and in-app notification to all targeted attendees. Automated reminders go out again **3 days before** and **1 day before** the event.
  </Step>
</Steps>

<Tip>
  Use targeted attendees instead of school-wide when an event only concerns a specific class. For example, for a Grade 7 parents' evening, add only Grade 7 guardians and Grade 7 teachers as attendees — everyone else won't receive a notification.
</Tip>

## Recurring events

Recurring events let you define a repeating schedule once. When you mark an event as recurring, Elimu Bora generates individual instances based on the recurrence rule you set. Each instance sends its own reminders. This is useful for events like:

- Weekly Monday assemblies
- Monthly staff meetings
- Bi-weekly sports practice

## Attachments

The attachments feature connects an event to existing data in Elimu Bora or to uploaded files:

- **Assessment attachment** — link the "End of Term Exam" assessment to your exam-week event so teachers and students can see the exam details directly from the event.
- **Invoice attachment** — link a fee invoice to a "School Fees Day" event to remind guardians of an outstanding balance.
- **File uploads** — upload circulars, timetables, or event programmes as PDF or document files. Attendees can download them from the event detail page.

## Notifications

When you create an event, all targeted attendees receive:

- An **in-app notification** (visible via the bell icon in the top navigation)
- An **email notification** to their registered email address

Automated reminders are sent at **3 days before** and **1 day before** the event date. No manual follow-up is needed.
