Create accounts for anyone who needs access to Elimu Bora. The role you assign determines what they can do.

## How to add a user

1. Go to **Users** in the sidebar.
2. Click **New user**.
3. Choose the user type: **Staff**, **Student**, or **Guardian**.
4. Fill in the required fields (full name, phone, email, ID number).
5. Select a **Role**.
6. Click **Save**.

The user receives a welcome email with their temporary password.

## Notes

- Email is required for login. Use a personal address for guardians.
- Staff records can include department and assigned classes.
- Students need an admission number — set it before saving.

## Troubleshooting

- "Email already exists"? That email is in use — search Users first.
- User can't log in? Resend the welcome email from the user profile.
